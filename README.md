## 📱 Mini App: Counter App
I built a simple interactive counter app as part of my learning.

### 🚀 Features
- Increase (+1) and decrease (-1) counter
- Reset counter
- Prevent negative values
- Disable decrement button at zero
- Persist counter using localStorage
- Dark mode toggle 🌙
- Save dark mode preference
- Clean card-style UI

### 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

### 📦 How to Run
1. Open the `mini-app` folder  
2. Open `index.html` in your browser
